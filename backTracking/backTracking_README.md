####5.求数组中和为给定数所有的组合

给定一个数t，以及n个整数，在这n个数中找到加和为t的所有组合，例如t=4，n=6这6个数为[4,3,2,2,1,1]，这样输出就有4个不同的组合它们的加和为4:4,3+1,2+2,and 2+1+1。请设计一个高效算法实现这个需求。

解题思路：先将数据按从大到小进行排序，然后使用回溯法遍历所有可能。注意去掉重复的结果。
